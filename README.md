# wallet
A simple proof of concept of a banking system using spring boot and hexagonal architecture

## Basis of the assumption

Rest API to simulate a small bank:

- User registration
- Account creation (wallet)
- Realization of money deposit
- Account display (wallet) --> Balance and movements
- Transfer from account A to account B

## Technologies

- Spring Boot (Data JPA, Web, Validation, Test)
- H2
- Lombok
- Mapstruct
- Swagger
- Jacoco
